# 受け入れテスト

## A. 起動と権限

- [ ] 初回起動時に製品目的が分かる。
- [ ] カメラを拒否してもフリー画像または生成画像で続行できる。
- [ ] マイクを拒否した場合に復帰方法が分かる。
- [ ] センサー権限を拒否しても基本トレーニングが動く。
- [ ] 権限の利用目的が要求前に表示される。

## B. 人物除外

- [ ] 人物を撮影しない注意が常時確認できる。
- [ ] 顔検出対応端末では顔を検出した画像を拒否する。
- [ ] 顔検出非対応端末では機能を保証しない旨を表示する。
- [ ] フリー画像検索で人物関連タグを除外する。

## C. 教材

- [ ] カメラ撮影からお題表示へ進める。
- [ ] 4回ごとに同梱写真3点とWikimedia Commons画像1点が順不同で選ばれる。
- [ ] Commons画像の作者・ライセンス・原典が表示される。
- [ ] Commons取得失敗時に同梱写真へ切り替わる。

## D. 回答

- [ ] 音声認識開始が視覚的に分かる。
- [ ] 中間認識結果が更新される。
- [ ] 15秒で自動終了する。
- [ ] 無回答でも操作不能にならない。
- [ ] 連続回答でタイマーが重複しない。

## E. 採点

- [ ] 10軸の点数が各0〜10点に収まる。
- [ ] 合計が0〜100点に収まる。
- [ ] 同条件の同回答で点数が一致する。
- [ ] 採点バージョンを履歴に保存できる。
- [ ] 完全な実名が画面・ログ・保存データへ出ない。
- [ ] 各軸の根拠を展開できる。

## F. トレーニング

- [ ] 最弱能力に対応した改善ヒントが出る。
- [ ] 同じ写真へ「この写真で一言。」で再挑戦できる。
- [ ] 前回の得点を再挑戦中に保持する。
- [ ] 結果に得点差が表示される。
- [ ] 新しい教材へ移ると比較基準がリセットされる。

## G. オフライン

- [ ] 初回オンライン起動後、機内モードで起動できる。
- [ ] オフライン生成画像を使える。
- [ ] お題、採点、ヒント、再挑戦が完走する。
- [ ] オンライン復帰後にアプリが壊れない。

## H. プライバシー

- [ ] 生の画像をIndexedDB・LocalStorageへ保存しない。
- [ ] 生の音声を保存しない。
- [ ] 位置情報を使用しない設定が可能である。
- [ ] 履歴削除操作がある。
- [ ] データ用途をアプリ内で確認できる。
